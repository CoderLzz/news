# news

### 项目介绍

该项目是一个新闻资讯后台管理系统，可以对注册用户、新闻资讯以及前端网站进行管理，新闻资讯的前端网站是我另一个项目news_web

### 技术

- 使用vue-cli 4 脚手架搭建的vue项目
- vue全家桶
- axios
- 国际化(vue-i18n)
- element-ui和iview
- echarts
- nprogress
- font-awesome
- vue-quill-editor
- less

### 使用

- 下载依赖

  ```
  npm install
  ```

- 运行

  ```
  npm run serve
  ```

- 打包

  ```
  npm run build
  ```
